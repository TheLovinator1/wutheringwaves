During the event, 5-Star Resonators: Jiyan, Yinlin, Zhezhi, Xiangli Yao, Phoebe, and 4-Star Resonators: Lumi, Taoqi, Yuanwu receive boosted drop rates!

Duration

Version 2.3 update - 2025-05-22 09:59 (server time)

Eligibility

Reach Union Level 8 or complete the combat trial in ""First Resonance"" and activate the Convene function.

Convene Rules

  • This is an Anniversary Resonator Convene event banner. Use Radiant Tide to convene. You are guaranteed at least one 4-Star or higher rarity Resonator or Weapon every 10 attempts.

  • The Guarantee count is shared among all [Anniversary Resonator Convene] and [Featured Resonator Convene] event banners. If a 5-Star Resonator is not obtained in this event banner, the Guarantee count is carried over across other [Anniversary Resonator Convene] and [Featured Resonator Convene] event banners and resets once a 5-Star Resonator is obtained.

  • You can select one 5-Star Resonator within a designated range as the Featured Resonator of the event banner (you can re-select at any time). Every time a 5-Star Resonator is obtained, there is a 50.00000% chance it will be the Featured 5-Star Resonator of this event banner, and a 50.00000% chance it will be a random non-Featured 5-Star Resonator (not including the selectable Resonators for this event banner). All 5-Star non-Featured Resonators have equal drop rate. If the 5-Star Resonator obtained is not the Featured Resonator, the next 5-Star Resonator is guaranteed to be the Featured Resonator.

  • Selectable 5-Star Resonators: Jiyan, Yinlin, Zhezhi, Xiangli Yao, Phoebe

  • The selectable 5-Star Resonators' Wavebands will become available in the Afterglow Coral Store, each with a limit of 2 purchases, and they will be removed once the Anniversary Resonator Convene event ends.

Reminder: Rewards for the 5-Star Trial Resonator event associated with [Anniversary Resonator Convene: Phase I] will be sent via in-game mail.

Eligibility: Create a character and unlock the Mails feature before 2025-04-29 04:00 (UTC+8). The rewards will remain claimable until 2025-05-22 09:59 (server time).

For more details, please refer to the in-game Convene page.
