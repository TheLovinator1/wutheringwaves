Game Launcher FAQ

Q1: The Game won't run after installation. A prompt says: "vcruntime140_1dll was not found".

A1: Please reinstall the Launcher and download the Client again. If this happens repeatedly, please follow the instructions here to install these files before running the Launcher:

a) Error when installing the Launcher:

Download this file: aka.ms/vs/17/release/vc_redist.x86.exe

Install the program.

Reinstall the Launcher.

b) Error when running the Launcher:

Download this file: aka.ms/vs/17/release/vc_redist.x64.exe

Install the program.

Reinstall the Launcher.

Q2: My screen went blank after I started the game, and when I clicked on somewhere a pop-up would say: "Program not responding".

A2: To troubleshoot this issue, please try the possible solutions below:

  • Check if your device meets the specs listed above.

  • Add the game to your malware protection programs' whitelists, then restart the game.

  • Shut down the program and restart the game.

  • Find launcher.exe in the game's installation directory. Right click→Properties→Compatibility Tab→Check the box that says "Run this program in compatibility mode"→Apply and OK. Then restart the game.

  • Update your graphic driver to the latest version, then restart the game.

  • Click on the "Repair" button in the top right corner to verify and repair the game's files.

If the issue persists, contact our Customer Service.

Q3: When I download the game, the download speed is slow/unstable.

A3: The download speed is subject to your internet connection and the number of concurrent downloads. Please wait patiently.

Q4: The download speed went to zero and the download was stopped.

A4: Please make sure your internet connection is working. If the download doesn't start, try pausing and resuming the download later. If this issue persists, contact our Customer Service.

Q5: My malware protection program says "dll files being modified" during the download.

A5: All downloaded content is essential for the game to function properly. If your anti-malware blocks the download, try dismissing the prompt or temporarily disabling your protection to re-download the game safely.

Q6: The game crashes or exits immediately after starting up.

A6: You can try clicking on the "Repair" button in the upper right corner of the launcher to repair the files. This might take a while. Please be patient before it is completed.

Q7: The launcher indicates "Repair Complete", but the game still crashes or exits immediately after starting up.

A7: Please restart your computer and try repairing it again. If this issue persists, please contact our Customer Service for assistance.
