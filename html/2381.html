Features

**

Profile**

New feature: Titles

  • View or change your Title via Terminal> Profile > Change Title

  • Available Titles and their respective Unlock requirements are displayed on the Change Title page.

  • Challenge Records before and after certain Titles are obtained will be displayed on the page. The Records will be updated if new ones are achieved.

  • More Titles will be added in future versions.

**

Story**

New feature: Story Review

  • During a scene, press the icon in the top right corner of the screen to access Story Review.

  • This feature allows you to replay previously voiced lines and view their subtitles.

  • To access more storyline content, go to Terminal > Gallery > Main Story.

**

Map**

  • Added a "Switch Gravity" view option to the Map for Avinoleum.

  • Added Switch Gravity-related Markers for Avinoleum.

  • Added icons on the Map to show certain undiscovered gameplay content.

  • Added the option to Track and Fast Travel to Collection Spots and locations marked with Lootmappers or Custom Markers.

  • Adjusted the order of Regions displayed when "Switch Map" is pressed. Newer Regions will be displayed first.

Adjusted the rules under which certain red dots appear on the Map. A red dot will still appear if there are unclaimed rewards.

Graphics & Performance

**

Graphics Settings**

Added Color Settings

  • Adjust the Color Settings via Settings > Graphics > Basic > Color Settings

  • The options "Saturation" and "Contrast" are now available, in addition to "Brightness."

  • Images are provided for reference to show the effects of different Settings.

Please note that the Brightness value will be reset to the default 50 following the Version 2.2 update. If you set a Brightness value that is different from this default option, please readjust the value accordingly following the update.

Added notices regarding Frame Generation settings

  • If Hardware-Accelerated GPU Scheduling is not enabled on Windows, the "Frame Generation" option will be reset to "Off" following the Version 2.2 update. To turn on the Frame Generation option, enable the "Hardware-Accelerated GPU Scheduling" option via "Settings > System > Display > Graphics Settings" on Windows and then reboot your PC.

  • In the current version, the Frame Generation option is available for devices equipped with the NVIDIA RTX 40 series or later.

Added an On/Off option for Co-op Teammate VFX

Turn this feature on or off via Settings > Graphics > Custom

  • Added physics effects to fabric, making fabric in certain scenes appear more realistic.

  • Added the 120 FPS option for certain Mac models. This option is now available for Mac devices equipped with M3 Pro, M3 Max, M4, or M4 Pro processors.

  • Optimized Ray Tracing in certain scenes.

  • Optimized the efficiency of the shader compilation process and the hardware resource usage on PC.

  • Optimized the visual effects for fog and cloud under certain settings on PC.

Other

**

Title Screen**

New feature: Network Test

  • If you encounter network-related issues, such as failure to log in, you now have the option of performing a network test via Tools > Network Test on the title screen.

  • The network testing process may take some time.

  • If needed, you can report the network testing results and contact Customer Service to help us better identify the issue.

  • Optimized the log-in process and the process of entering or exiting the title screen, reducing the time required to log into and enter the game.

  • Increased the number of Waypoints that can be deployed simultaneously to 10.

  • Increased the number of Casket Sonar that can be deployed simultaneously to 5.

  • Optimized the descriptions of certain in-game settings and functions.
