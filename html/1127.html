Dear Rover,

To provide a better gaming experience for all Rovers, we will begin a grayscale test of the pre-download feature for the official launcher in version 1.2.

Once pre-download is available, Rovers participating in the grayscale test can use this feature to download version 1.2 update package in advance.

With pre-download, Rovers can experience the new version more quickly after the maintenance.

To ensure stability and compatibility, a group of Rovers using the official PC client will be randomly chosen to join the grayscale test. The pre-download feature will be available to all Rovers through the official launcher in future updates. Please stay tuned to our official announcements for more details.

Pre-download Time

Starting from 2024-8-13 10:00 (UTC+8)

How to Pre-download

  1. Rovers selected for the gray box test will need to update their launcher to version 1.6.0.0 upon starting the game launcher. When prompted, please click "Confirm" to proceed with the update.

  2. After the pre-download begins, a [New Version Pre-download] button will show up on the v1.6.0.0 launcher's interface. Click this button to download the latest package.

Note: The pre-download package for PC is 23 GB in size. Please make sure you have enough storage space.
