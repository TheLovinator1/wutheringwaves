Adjustments and Optimizations

Resonators & Echoes

  • Optimized the positions of Phantasmic Imprints in complex terrain when Resonator Zhezhi casts her Resonance Skill, reducing potential glitches in the performance of Zhezhi's model.

  • Optimized the movements of Resonator Zhezhi when casting certain Echo Skills or using the Levitator.

  • Optimized the display of damage numbers in combat for some Resonators.

Features

Data Bank:

Added the "Batch Merge" feature to Data Merge.

  • Use the "Batch Merge" feature to add in a large number of Echoes, up to 100 at a time.

Introduced a "Discarded" marking option for easier Echo management.

  • Mark an Echo as "Discarded" on the Echoes screen.

  • Select "Show Discarded First" in the sorting menu to prioritize marked Echoes.

Note: Marking an Echo as "Discarded" will automatically remove its locked status, if applicable.

Map:

Added a "Switch Map" feature

  • Quickly switch between different map areas on the "Switch Map" page.

  • Press the button in the bottom right corner to zoom in on your current location.

Introduced a "Fast Travel" feature

  • Teleport to an activated Resonance Beacon near your destination, then start tracking.

Added a "Collection Spots" feature for Ascension Materials tracking

  • Select the icon for an Ascension Material on the Backpack or Resonators page, then select "Collection Spots" to view its location on the map.

  • Once all of this particular material has been collected from the current location, the next location will automatically become trackable.

Story:

Added a "Summary" feature

  • When skipping certain story content, a summary of the skipped parts will display, allowing you to follow the story's progress.

Gallery:

Added an "Archives" section on the Gallery page.

Leveling

Item Exchange:

New items have been added to the Item Exchange page's Oscillated Coral selection:

  • MF Whisperin Core x3, MF Howler Core x3, and Basic Ring x3. Up to 40 exchanges for any of these items.

  • The refresh interval for the new items is the same as the existing ones in the Oscillated Coral selection.

Intimacy:

Adjusted the distribution of Intimacy points obtained upon claiming rewards after completing of a challenge. Intimacy points will now be granted to the Resonators who participated in the challenge.

Controls

Controller Support:

Wuthering Waves now supports specific controllers on mobile devices.

  • Wireless Controllers: Xbox series, Xbox Elite series, DualShock®4, DualSense®, DualSense Edge™

  • With the controller connected, press any button on the controller while the main interface is displayed, and a Controller Mode-related message will pop up.

  • You can also switch to or exit Controller Mode manually via Settings while the controller is connected.

Optimized certain functions in Controller mode

  • An additional icon appears after activating a skill using a combination of buttons in Controller mode.

  • The touchpad on a Sony controller is supported in Controller mode, functioning as a "Share" key.

  • For certain dialogue scenes with adjustable angles, angle adjustments are now possible in Controller mode.

Keyboard & Mouse Mode:

Added a "Quick Access Wheel" feature to the Keyboard & Mouse mode on PC

  • In the default key configuration setting, press and hold "CapsLock" while on the main page to bring up the Quick Access Wheel.

  • Move the cursor over the icon for the page you wish to jump to, then release "CapsLock" to navigate to that page.

  • Press and hold the mouse wheel while bringing up the Quick Access Wheel to edit it.

Key & Button Setting Customization:

  • Customized key or button settings can be synchronized between devices in Touchscreen, Keyboard & Mouse, or Controller mode

  • Button settings in Touchscreen mode can be synchronized between mobile devices.

  • Key settings in Keyboard & Mouse mode can be synchronized between PCs.

  • Button settings in Controller mode can be synchronized between mobile devices and PCs.

After updating to Version 1.3, the default controls will match your initial choice between Keyboard & Mouse or Controller on PC. Subsequent changes to the key settings can be synchronized.

  • Adjusted the conditions for deactivating auto-targeting during combat across all modes. Auto-targeting will be deactivated when the camera movement speed reaches a certain threshold.
