[Adjustments and Optimizations]

Resonators & Weapons

  • Added the On/Off option for injury effects, accessible via Settings > Graphics > Custom.

Features

Echoes:

New Feature: Echo Suggestion

  • Access the Echo Suggestion feature through the character’s Echo Assembly interface to view recommended Echoes for the current character.

  • If multiple Sonata Effects are recommended, the usage percentage for each Resonance effect will be displayed separately.

  • From the Echo Suggestion interface, you can navigate to the Sonata Gallery to view more detailed information about Echo sets.

New Feature: Mainstat & Substat Filtering

  • In the Echo Equipment details interface, you can now filter combinations of Mainstat and Substat attributes based on your needs.

New Feature: Echo Preset

  • Access the Echo Preset feature by clicking the Echo Preset icon in the bottom-right corner of the character’s Echo Assembly interface.

  • Select the Current Setup column and use Save As Preset to save the equipped Echo lineup for the Resonator. After entering a preset name, it will be saved as an Echo Preset. Select any saved preset and use Apply to equip it to the current Resonator.

  • Expand the attribute comparison interface to directly compare the differences between the Current Setup and other Echo Presets.

  • Up to 20 Echo Preset configurations can be saved.

  • Increased the storage limit for Backpack - Echoes to 3,000.

Camera Mode:

New Feature: Filter

  • In Camera mode, apply a filter (e.g. Vivid, Dystopian, Film, Noir) to your photos.

  • You can also adjust the intensity of said filters.

-New poses are added for certain Resonators to offer you more options when taking photos.

More posing styles will be made available in future versions. Please stay tuned.

Graphics & Performance

  • Added the [120fps] option for devices equipped with specific graphics cards. Users with NVIDIA RTX 50 Series graphics cards can enable this option via Settings > Graphics > Frame Rate.

  • Added Ray Tracing for devices with NVIDIA RTX 40 Series graphics cards. Choose "Low," "Medium," or "High" in Settings > Graphics > Ray Tracing, or select specific effects like "Ray-Traced Reflections" or "Ray-Traced Global Illumination."

  • Added DLSS Frame Generation for devices with NVIDIA RTX 40 Series graphics cards. Access this feature via Settings > Graphics > Advanced > Frame Generation. Enable this option to generate additional frames.

After enabling Frame Generation, ensure that the "Hardware-Accelerated GPU Scheduling" option is enabled in "Settings > System > Display > Graphics Settings" on Windows before rebooting your PC. If you encounter performance issues such as FPS drops or stuttering, verify that the "Hardware-Accelerated GPU Scheduling" option is enabled. If the issue persists, contact Customer Support for assistance.

  • Upgraded NVIDIA DLSS to Version 3.7, offering improved Super Resolution and Frame Generation effects when enabled. The "Settings > Graphics > Super Resolution" option has been reset to "Auto."

The development team is committed to ongoing optimization efforts to expand support for these features to additional graphics cards (including other NVIDIA RTX Series and AMD models). Please stay tuned for official announcements regarding the availability of these features for other graphics cards.
