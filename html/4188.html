[Adjustments and Optimizations]

Resonators & Weapons

**

Resonators' Movements**

Optimized Resonators' Movements

  • Basic movements: walking, running, jumping, and stopping.

  • Advanced movements: climbing, gliding, flying, moving up and down stairs, and getting hit.

  • In Version 3.1, the above movements, along with the transition between different movements, will be smoother.

  • Resonators with different body types move slightly differently.

  • Optimized the visuals when certain skills hit targets for some Resonators.

Features

**

Expedition Motorbike:**

Added Glacier Track

  • As you progress through the Main Quests, the Glacier Track ability will be unlocked for the expedition motorbike, allowing it to drive across open water.

Added "All-Terrain Tactics" Expansion Module

  • As you progress through the Main Quests, new Expansion Modules, which provide various buffs, will be unlocked for the expedition motorbike. The "All-Terrain Tactics" Expansion Module will be unlocked in Version 3.1's Main Quest.

  • To switch between Expansion Modules, go to the Motorbike Expansion Module page or the Expansion Module Switch of the Vehicle section of the Utility wheel. Only one Expansion Module can be active at a time.

  • Core Module remains effective when switching between different Expansion Modules, while Combat Modules change with the selected Expansion Module.

Added New Content to Motorbike Looks

  • Added two new motorbike cosmetics categories: Chassis Set and Motorbike Accessory.

  • Visit Expedition Motorbike > Motorbike Appearance to make adjustments to the motorbike looks.

Optimized Autopilot Mode

  • When navigating across the map, you can use Fast Travel to teleport to the nearest beacon that responds to Autopilot Mode and activate Autopilot Mode.

Optimized Adaptive Hook

  • Adjusted the trigger order concerning Adaptive Hook. Adaptive Hook will be triggered, instead of Autopilot Mode, when Adaptive Hook is available.

  • Minimized the incidence of triggering another function accidentally when using Adaptive Hook multiple times in succession.

  • Added an HUD display of the Expedition Motorbike's dashboard in the first-person view.

  • Optimized the visual presentation and color display of the Expedition Motorbike's dashboard.

**

Starpaths Intertwined:**

Added Starpaths Intertwined Giveback Event

  • Complete Starpath Tasks to earn Journey Points. Reach Journey Point milestones to claim Starpath Gifts.

  • You can obtain Journey Points by logging in to the game, obtaining 5-Star Resonators/weapons, and obtaining Trophies.

  • There is a Journey Point limit for each Starpath Task. Once the limit is reached, no additional points can be obtained from that task.

  • The Starpaths Intertwined menu will pop up when you log in to the game after Version 3.1 goes live. Or alternatively, you can access it via Terminal > Profile > Starpaths Intertwined.

For more details about the rules, please visit the in-game Starpaths Intertwined page when Version 3.1 goes live.

**

Material Tracking**

Optimized Material Acquisition Process

  • Clicking on a material via the Backpack or Resonators screen and selecting the "Tacet Discord" acquisition path will now directly track the Tacet Discords that drop the material on the map, instead of jumping to the Enemy Tracing menu.

  • When the tracked Tacet Discords are defeated, the next group of Tacet Discords will be automatically tracked.

Leveling

**

Data Bank:**

Added Targeted Merge

  • This feature unlocks at Data Bank Lv. 15. Upon unlocking it, you can access it via Data Bank > Data Merge.

  • You can merge 5 un-upgraded 5-Star Echoes into a random new Echo of the Sonata Effect of your choice. Targeted Merge does not produce extra Echoes.

  • Targeted Merge can be used up to 10 times per week, and resets at 04:00 (server time) every Monday.

In Targeted Merge, you can't select a Sonata Effect for which no Cost-1 or Cost-3 Echo has been unlocked.

Added Substat Data Modification

  • Data Modify unlocks at Data Bank Level 21. Upon unlocking it, you can access it via Data Bank > Data Modify and start Modifying Substats.

  • Use a Transducer to adjust the Substats of a selected Echo to random stats, excluding locked Substats. This can only be done on fully tuned 5-star Echoes.

  • The number of locked Substats determines the amount of Transducers consumed in each Modification: With 2 or fewer locked stats, consume 1 Transducer. With 3 locked stats, consume 2 Transducers. With 4 locked stats, consume 3 Transducers.

  • You may choose whether to apply the Modification results in Substat Modification. Discarding the result does not refund used Transducers.

Substat Modification results that are not confirmed due to unexpected errors will be saved in the server for up to 30 days. Please confirm your results in time.

Adjusted Mainstat Data Modification

  • Added batch modification for Echo Mainstats.

  • Up to 10 Echoes can be selected in a single Mainstat Data Modification attempt.

Graphics & Performance

**

PC Graphics**

Added Display of Device Info on PC

  • View the information of the current device via Settings > Graphics > Device Info.

  • The information includes CPU, Memory, Graphics Card, and other hardware info, as well as Graphics Driver version and other runtime environment info.

  • This information may help you better identify the cause of issues that affect the smooth running of the game.

Extended Ray Tracing to Some PC Devices Equipped with AMD

  • Ray Tracing is supported in Version 3.1 on devices equipped with RX 9000 series graphics cards.

Please note that the Graphics Driver needs to be updated to Adrenalin 25.10.2 or newer in order for Ray Tracing to take effect when enabled.

  • Added interaction effects to snow and specific types of vegetation in Roya Frostlands: Frostlands Surface on PC.

  • Optimized visual effects of the interaction with petal piles in Chronorift Metropolis on PC.

  • Adjusted the Capsule AO options on PC. Three options are now available: Low, Medium, and High, instead of two (On/Off).

  • Adjusted the default graphics settings on PC. Lower settings will now be selected by default for some entries to better fit the hardware performance of the device.

Note: HDR support, originally planned for release in Version 3.1, will not be available in this upcoming version, as it has not yet met the required standard. We sincerely apologize for any inconvenience you may experience. Please stay tuned for official announcements regarding the availability of HDR support. Thank you for your support and understanding.

**

Mobile Graphics**

Extended Ultra High Resolution Option to More Mobile Devices

  • The Ultra High resolution option will be available for more mobile devices in Version 3.1. To enable this option, the processor must meet one of the following requirements:

  1. Snapdragon 8 Gen 3 or newer

  2. MediaTek Dimensity 8400 or newer, MediaTek Dimensity 9300 or newer

  3. Exynos 2500 or newer

  • To enable this option, go to Settings > Graphics

  • Enabling this option will increase the clarity of in-game visuals.

Due to variations in the in-game settings and system environment, enabling the option may occasionally cause performance issues, in-game glitches, and hardware-related issues. If you encounter any issue, please try disabling the option. If the issue persists, please contact customer support for further assistance.
